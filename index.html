--- title: Overview ---

Yesod Web Framework is meant for creating RESTful, type-safe, performant web applications in Haskell. It allows you use the compiler to help you program solid sites.

The current release is 0.0.0; the upcoming release is 0.2.0.

Unlike many other frameworks, Yesod includes very little functionality in its own package. Most useful functionality- templating, controller, etc- are provided by external packages. Many of these packages were created in the process of writing Yesod. However, they are provided separately so that they are available for use in other code. The packages used for different functionality is:

Component Yesod 0.0.0 Yesod 0.2.0
Server interface WAI WAI
Controller Quasi-quoted interface, included in Yesod web-routes-quasi
View HStringTemplate hamlet
Model No support persistent

Documentation is sparse at the moment while efforts are put towards releasing the new release.
